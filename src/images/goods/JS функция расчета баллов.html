Вот пример функции на JavaScript, которая решает данную задачу:

javascript
function calculatePoints(tasks) {
  const numStudents = tasks.length;
  const numTasks = tasks[0].length;
  const points = [];

  for (let i = 0; i < numStudents; i++) {
    let numFailed = 0;
    for (let j = 0; j < numTasks; j++) {
      if (tasks[i][j] === 0) {
        numFailed++;
      }
    }
    const score = numFailed === 0 ? 1 : numFailed + 1;
    points.push(score);
  }

  return points;
}

Функция принимает двумерный массив tasks, в котором на первом уровне вложенности находятся ученики, а на втором - задачи. Значения в массиве равны 0, если ученик не решил задачу, и 1, если решил.

Для решения задачи функция сначала определяет количество учеников и задач. Затем она проходится по каждому ученику и подсчитывает количество не решенных им задач. Если все задачи решены, ученик получает 1 балл, иначе он получает количество не решенных задач плюс 1.

Результат вычислений возвращается в виде массива points, где каждый элемент соответствует баллам, полученным одним из учеников.

Если вы хотите найти индекс подмассива в другом массиве, вам необходимо написать свою собственную функцию, которая будет проходить по элементам массива и сравнивать их с подмассивом. Например, вы можете использовать цикл for и метод every() для проверки, все ли элементы подмассива находятся в другом массиве, и затем вернуть индекс подмассива:

javascript
function findSubarrayIndex(arr, subArr) {
  for (let i = 0; i < arr.length; i++) {
    if (arr[i].length !== subArr.length) continue;
    if (arr[i].every((value, index) => value === subArr[index])) return i;
  }
  return -1;
}

Эта функция проходит по каждому элементу массива arr и проверяет, имеет ли элемент такую же длину, что и subArr. Если это не так, функция переходит к следующему элементу. Если элемент имеет такую же длину, как и subArr, функция использует метод every() для проверки, что каждый элемент subArr равен элементу с тем же индексом в arr. Если это верно для всех элементов, функция возвращает индекс элемента в массиве arr, в котором находится subArr. Если ни один элемент не соответствует критериям, функция возвращает -1.

Вы можете вызвать эту функцию следующим образом:

javascript
const arr = [[1, 2, 3], [4, 5, 6], [7, 8, 9]];
const subArr = [4, 5, 6];

const index = findSubarrayIndex(arr, subArr);
console.log(index); // 1

В этом примере функция найдет subArr в массиве arr и вернет индекс 1, потому что subArr совпадает со вторым элементом в массиве.

function countPoints(arr) {

    let count = 0;

Для решения этой задачи можно использовать следующий алгоритм:

  1. Создать пустой массив, в котором будут храниться баллы каждого ученика.
  2. Для каждого ученика посчитать количество не решенных задач (количество нулей в его массиве) и сохранить это число.
  3. Пройти по массиву учеников и для каждого ученика посчитать, сколько баллов он заработал. Для этого нужно вычислить количество остальных учеников, которые решили данную задачу и добавить к итоговому баллу ученика это количество. Например, если только один ученик не решил данную задачу, то к его итоговому баллу нужно добавить 1. Если два ученика не решили данную задачу, то к итоговому баллу нужно добавить 2 и так далее.
  4. Добавить итоговый балл каждого ученика в массив баллов и вернуть этот массив.

Вот реализация данного алгоритма на JavaScript:

javascript
function calculateScores(tasks) {
  const scores = [];

  // Подсчет количества нерешенных задач для каждого ученика
  const numUnsolved = tasks.map((studentTasks) => studentTasks.filter((task) => task === 0).length);

  // Подсчет итоговых баллов для каждого ученика
  for (let i = 0; i < tasks.length; i++) {
    let score = 0;
    for (let j = 0; j < tasks[i].length; j++) {
      if (tasks[i][j] === 1) {
        let numOthersSolved = 0;
        for (let k = 0; k < tasks.length; k++) {
          if (tasks[k][j] === 1 && k !== i) {
            numOthersSolved++;
          }
        }
        score += 1 + numOthersSolved * (numUnsolved[i] + 1);
      }
    }
    scores.push(score);
  }

  return scores;
}
